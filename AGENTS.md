# AGENTS.md

수치해석 수업 실습·과제 저장소(근 찾기 수치해법 구현). 범용 코딩 에이전트를 위한 안내 문서.

## 명령어

- 실행(.py): `python "algorithms/<파일>.py"` — 함수·초기값·TOL이 스크립트 상단에 하드코딩됨
- 실습(.ipynb): Jupyter 또는 VS Code에서 셀 실행
- 설치: `pip install numpy scipy matplotlib` (표준 라이브러리 `math` 포함)
- 테스트: 별도 프레임워크 없음

## 구조

- `algorithms/` — 이분법·가위치법·고정점 반복·뉴턴법(중근)·할선법 구현 `.py`
- `exercises/` — 교재 장별 연습·과제 노트북

## 컨벤션

- 알고리즘 스크립트는 상단에서 `f(x)`, 구간/초기값, `TOL`을 정의하고 while 루프로 반복하며 중간값을 리스트에 누적
- 파일명은 알고리즘명(영문) 또는 교재 문제 번호. 한글·공백 포함 가능

## 주의

- 초기 구간·초기값이 예제별로 하드코딩됨. 재사용 시 `f`·범위·TOL을 함께 수정(잘못된 구간은 발산/무한루프)
- 뉴턴·할선법은 도함수/이전 점도 수기 작성이라 `f`만 바꾸면 안 됨
- 한글 파일명 NFC 통일, `.ipynb_checkpoints/`·`__pycache__/`·`.DS_Store` 커밋 금지
- CLAUDE.md와 내용 동일하게 유지
